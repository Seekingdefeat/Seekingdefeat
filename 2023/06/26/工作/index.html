变强

工作方面

心态/情绪

  • 0.1最小成功法

    • 理性对待每一个任务/Bug，可能会比较难做，情绪上也有些为难
    • 但是慢慢一步一步分析+拆分，拆分成可以执行并且可以获得成功的最小单位
    • 一个一个分析解决以此获得成功的感觉，情绪上也就有了积极主动的能动性，效率自然就高起来了。

  • x + y = z

    • 360°，每一件事情/每个项任务都有多个角度，一个角度不行，就思考换一个角度去解决，并且不断优化迭代记录，争取找到最短路径
    • 从变量上入手
      • 添加变量 x + y + k
      • 改变变量 m + y
      • 删除某个变量
    • 从算法上入手/换方法
      • x + y -> x * y
      • x * y -> x 的 y次方
    • 从结果上入手
      • 是不是必须达到z？变换结果 z -> z1
      • 否定结果z -> 直接舍弃

  • 付出/回报

    • 能量守恒
      • 付出了3分的能量，就别期望太高，想要获得5分的回报
      • 脚踏实地，一步一步做，一点一点改变/成长
      • 情绪上别急/焦躁，因为没什么用。凡事一步一步来，别老想着一步登天，不切实际。
    • 总结规律-抽象
      • 天道酬勤 -> 天道 ~~自然的规律 酬勤 ~~ 并不是奖励那些一直努力的人，而是奖励那些善于总结/归纳/抽象 大自然规律(包括做事规律) 的人。
      • 上述两者都可以算是努力，但是一个骑自行车，一辈子都是骑自行车，而后者可以通过方法/思维的提炼抽象，日日进，突飞猛进。

  • 梦想

    • 梦想是什么？
      • 梦想就是你非要达成的目标，不达成/不让你去走在实现他的路上就还不如让我去死的信念
      • 非干不可，人生活给自己的重大意义
    • 抬头看路
      • 不要固步自封，不要因为眼前的困难而遮蔽了/忘却了你想要到达的目的地
      • 克服困难/行动力拉满的上层建筑
    • 我的梦想是什么？
      • 我的梦想是：让全世界的人都运行我写的代码
    • 代码是什么？
      • 代码是一种艺术，当一名艺术家，而不是程序员/码农

  • 追求本质

    • 万事万物争取向上抽象/why?
      • 抽象的东西可以概括众多具体 eg：阴阳可以概括99%的世间万物
      • 抽象可以自由实现，灵活
      • 抽象代表着一种思想的提炼，类似于树木的根部，从而衍生出很多树枝/树叶
